# RB Gen

A minimalist Rubik's Cube scrambler and timer built with Godot.

<img width="1887" height="1033" alt="Screenshot 2026-06-21 190246" src="https://github.com/user-attachments/assets/fdd86bb7-72e2-4058-9baf-c0365c1944b8" />



## Features

* Random scramble generation
* Solve timer (space key)
* Session time tracking
* Minimalist interface

## Built With

* Godot Engine
* GDScript
